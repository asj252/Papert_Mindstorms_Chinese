Seymour Papert 教授在2016年7月31 日過世了，Papert 教授在歐美教育/科學界有著崇高的地位，當今世界各國制定教育政策的重要人士，或是諾貝爾獎/圖林獎得主，很多都是Papert的朋友或弟子，Papert 在世時（到2006為止，2006之後 Papert 重病在床）雖然有很大的影響力，比方說推動了著名的100元電腦計劃，然而依舊時常感到，現存的學校制度頑強抗拒他提倡的改變，以至於無法將他的教育理念真正普及開來。

我們今天在此重讀 Papert 教授35年前的巨著 Mindstorms （1981）(樂高機器人名字取自於此)，有著雙重意義，首先Papert 在此書中以編程/數學的綜合教學法為例重新闡述了瑞士心理學家 Piaget 的心理髮展理論，即解釋人類是如何思考以及學習的。再來我們放下書本觀察當下，將會發現當下的學校制度開始主動往 Papert 所希望的方向改變，而 Papert 的思想在成書35年後仍然是超前並且引領著我們。
